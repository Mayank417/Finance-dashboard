# 💰 Zorvyn Finance Dashboard

A simple web app to track income and expenses.

## 🚀 Features

* Add & manage transactions
* View balance, income, expenses
* Export data to CSV
* Download PDF reports
* Light/Dark mode

## 🛠️ Tech

* HTML
* Tailwind CSS
* JavaScript

## ▶️ Run

Open `index.html` in your browser.
